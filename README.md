# 🌸 Flower Intelligence System

## Project Overview

The Flower Intelligence System is an Artificial Intelligence project that uses Machine Learning to classify flower species based on their physical measurements.

The system analyzes:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Using these measurements, the AI model predicts the flower species and provides additional information such as flower color, fragrance, lifespan, and description.

## Dataset

This project uses the Iris Dataset, one of the most popular datasets for Machine Learning classification tasks.

Flower Classes:

1. Setosa
2. Versicolor
3. Virginica

---

## Machine Learning Workflow

1. Load Dataset
2. Split Data into Training and Testing Sets
3. Train Decision Tree Classifier
4. Evaluate Model Accuracy
5. Predict Flower Species
6. Display Additional Flower Information
